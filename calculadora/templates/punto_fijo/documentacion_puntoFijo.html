Documentación teórica - Método de Punto Fijo

Objetivo del método

El objetivo principal del método de iteración es encontrar una solución aproximada para una ecuación, ya sea algebraica o transcendental, mediante un proceso iterativo.

Pasos del algoritmo

  1. Reescribir la ecuación:

    Transformar \( f(x) = 0 \) en \( x = g(x) \).

    Ejemplo: si \( f(x) = x^{2} - 2 \), se puede escribir como:

    \[ x = \sqrt{2} \quad \text{o} \quad x = \frac{x - (x^{2} - 2)}{2x} \]
  2. Elegir aproximación inicial \( x_0 \):

    Debe estar cerca del punto fijo verdadero, preferiblemente donde \( g(x) \) es continua y \( |g'(x)| < 1 \).

  3. Definir tolerancia \( \varepsilon \) y máximo de iteraciones \( N \):

    Ejemplo: \( \varepsilon = 0.0001 \), \( N = 50 \)

  4. Iterar:
    \[ x_{n + 1} = g(x_n) \]
  5. Calcular error:

    Error absoluto:

    \[ |x_{n + 1} - x_n| \]

    Error relativo:

    \[ \left| \frac{x_{n + 1} - x_n}{x_{n + 1}} \right| \]
  6. Verificar criterio de parada:

    Si Error \( < \varepsilon \), detener. Si se alcanza \( N \) iteraciones sin cumplir, también detener.

  7. Presentar resultado:

    Mostrar valor aproximado, iteraciones usadas y error final.

Ventajas

  • No necesita un intervalo, solo un punto inicial
  • Converge con rapidez (generalmente pocas iteraciones)
  • Alta precisión cuando converge
  • Fórmulas sencillas y fáciles de ejecutar
  • Proceso de resolución compacto y organizado

Desventajas

  • No garantiza convergencia para todas las funciones
  • Necesidad de construir funciones \( g(x) \) adecuadas
  • Puede ser difícil encontrar la \( g(x) \) correcta
  • Existen infinitas opciones para \( g(x) \) sin regla clara de selección
  • La convergencia depende de \( |g'(x)| \)

Ejemplo teórico

Problema:

Resolver la ecuación:

\[ x = \frac{1}{3}(x^{2} + 2) \]

La ecuación ya está en la forma adecuada:

\[ g(x) = \frac{1}{3}(x^{2} + 2) \]

Datos: \( x_0 = 1.5 \), \( \varepsilon = 0.001 \)

Fórmula de iteración:

\[ x_{n + 1} = \frac{1}{3}(x_n^{2} + 2) \]

Iteraciones:

n \( x_n \) \( x_{n+1} \) Error
0 1.500000 1.416667 0.0833
1 1.416667 1.393519 0.0231
2 1.393519 1.387486 0.0060
3 1.387486 1.385832 0.0017
4 1.385832 1.385368 0.0005 < 0.001 → Se detiene

Resultado:

\( x \approx 1.385 \)

Situaciones de uso

  1. Resolución de ecuaciones no lineales: Cuando no se puede resolver algebraicamente.
  2. Base de otros métodos numéricos: Como el método de Newton-Raphson.
  3. Simulaciones iterativas: En ingeniería eléctrica, mecánica y simulaciones físicas.
  4. Cálculo de raíces en modelos matemáticos: En economía, física, biología, etc.
  5. Optimización no lineal: Para encontrar mínimos o máximos transformando el problema.
  6. Computación científica: En análisis numérico de matrices y sistemas grandes.
