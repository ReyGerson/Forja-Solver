Documentación del Método Gráfico

Inicio

Objetivo del método

Solucionar problemas de programación lineal que se utiliza principalmente para casos con dos variables. Aunque no es muy práctico para una gran cantidad de variables, es muy útil para interpretar y analizar los resultados y la sensibilidad del problema.

Ventajas

  • Capacidad para encontrar la mejor solución: En situaciones donde los recursos son limitados.
  • Optimización de recursos: Permite maximizar el uso de recursos limitados, como tiempo, dinero y materiales.
  • Facilidad de implementación: Con el uso de software especializado, se puede implementar de manera eficiente en diversos escenarios.
  • Flexibilidad: Es posible ajustar los parámetros y restricciones para adaptarse a diferentes situaciones y necesidades.

Desventajas

  • Complejidad computacional: En algunos casos, resolver problemas puede requerir una gran cantidad de recursos computacionales.
  • Limitaciones en la modelización: Algunas situaciones del mundo real pueden no ajustarse perfectamente al modelo lineal, lo que puede llevar a soluciones subóptimas.
  • Sensibilidad a los datos de entrada: Pequeños cambios en los datos de entrada pueden tener un impacto significativo en los resultados, lo que requiere un análisis cuidadoso.

Situaciones de uso

Esta herramienta es ampliamente utilizada en campos como la economía, la ingeniería, la logística y más. Las situaciones más comunes donde se aplica el método gráfico son:

Optimización de productos

Determinar la combinación óptima de productos para maximizar beneficios.

Mezclas y formulaciones

Encontrar la combinación óptima de ingredientes o componentes.

Asignación de recursos

Distribuir recursos limitados entre diferentes actividades o departamentos.

Problemas de transporte

Optimizar rutas y costos de transporte.

Planificación de la producción

Programar la producción para maximizar la eficiencia.

Optimización de costos

Minimizar costos mientras se cumplen los requisitos de producción.

Pasos para realizar el algoritmo

1. Formulación del problema

Definir las variables de decisión, la función objetivo y las restricciones.

2. Gráfico de las restricciones

Representar gráficamente cada una de las restricciones en un sistema de coordenadas.

3. Identificación de la región factible

Determinar el área donde se satisfacen todas las restricciones simultáneamente.

4. Búsqueda de la solución óptima

Encontrar el punto (o puntos) en la región factible que maximiza o minimiza la función objetivo.

5. Análisis de la solución

Interpretar los resultados y verificar su validez en el contexto del problema.

Ejemplo

Paso 1: Formulación del problema

La fábrica de hilados y tejidos Salazar requiere fabricar dos tejidos de calidad diferente, estándar y premium; se dispone de:

  • 500 kg de hilo A
  • 300 kg de hilo B
  • 108 kg de hilo C

Requisitos por metro de tejido:

  • Estándar: 125 gr de A, 150 gr de B y 75 gr de C
  • Premium: 200 gr de A, 100 gr de B y 27 gr de C

Precios de venta:

  • Estándar: $4,000 por metro
  • Premium: $5,000 por metro

Variables:

X: cantidad de metros diarios de tejido tipo estándar Y: cantidad de metros diarios de tejido tipo premium

Restricciones:

0.12x + 0.2y ≤ 500 (kg de hilo A) 0.15x + 0.1y ≤ 300 (kg de hilo B) 0.075x + 0.027y ≤ 108 (kg de hilo C) x, y ≥ 0 (restricciones de no negatividad)

Función objetivo:

Zmax = 4000x + 5000y (máximos ingresos totales)

Paso 2: Gráfico de las restricciones

Igualamos las restricciones a 0 para encontrar los puntos de corte:

Restricción 1 (Hilo A):

0.12x + 0.2y = 500 Cuando x = 0 → y = 2500 → (0, 2500) Cuando y = 0 → x ≈ 4167 → (4167, 0)

Restricción 2 (Hilo B):

0.15x + 0.1y = 300 Cuando x = 0 → y = 3000 → (0, 3000) Cuando y = 0 → x = 2000 → (2000, 0)

Restricción 3 (Hilo C):

0.075x + 0.027y = 108 Cuando x = 0 → y ≈ 4000 → (0, 4000) Cuando y = 0 → x = 1440 → (1440, 0)

Paso 3: Identificación de la región factible

Con las coordenadas encontradas anteriormente:

  • Restricción 1 (Hilo A): (0, 2500) y (4167, 0)
  • Restricción 2 (Hilo B): (0, 3000) y (2000, 0)
  • Restricción 3 (Hilo C): (0, 4000) y (1440, 0)

Gráfico que muestra las restricciones y la región factible

Paso 4: Búsqueda de la solución óptima

Encontramos la intersección de las restricciones 1 y 2:

Ecuación 1: 0.12x + 0.2y = 500 Ecuación 2: 0.15x + 0.1y = 300 (multiplicamos por -2) Ecuación 2 modificada: -0.30x - 0.2y = -600 Sumamos 1 y 2 modificada: -0.18x = -100 Despejamos x: x = -100 / -0.18 ≈ 555.55

Reemplazamos x = 555.55 en la ecuación 1 para encontrar y:

0.12(555.55) + 0.2y = 500 66.666 + 0.2y = 500 0.2y = 500 - 66.666 = 433.334 y = 433.334 / 0.2 ≈ 2166.67

Paso 5: Análisis de la solución

La contribución obtenida (reemplazando las variables en la función objetivo) es de:

Zmax = 4000x + 5000y Zmax = 4000(555.55) + 5000(2166.67) Zmax ≈ $13,055,550

Solución óptima:

  • Tejido estándar (x): ≈ 556 metros
  • Tejido premium (y): ≈ 2167 metros
  • Máxima ganancia: ≈ $13,055,550

Esto significa que deben producir aproximadamente 556 metros de estándar y 2167 metros de premium para obtener la máxima ganancia.
