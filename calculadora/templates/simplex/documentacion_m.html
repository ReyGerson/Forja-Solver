Documentación del Método de la M

Inicio

Objetivo del método

El objetivo principal del método de la "M" es resolver problemas de programación lineal donde la solución inicial factible no está disponible, especialmente cuando se trata de restricciones de igualdad o desigualdad "mayor o igual que".

Ventajas

  • Es capaz de solucionar cualquier modelo, sin importar si el origen no pertenece a la región factible.
  • No existe un límite de variables.
  • Se basa en el algoritmo sencillo del método simplex.
  • Detecta cualquier tipo de solución del problema.

Desventajas

  • Al meter variables artificiales implica que el modelo incrementa en tamaño.
  • El manejo de la M puede hacer confuso u orilla a cometer errores.
  • Implica más pasos que en el método simplex.

Situaciones de uso

Restricciones de tipo "≥" (mayor o igual)

Cuando una restricción tiene el signo ≥, se resta una variable de holgura y se agrega una variable artificial para convertirla en ecuación.

Restricciones de tipo "=" (igualdad)

Las restricciones de igualdad no permiten utilizar variables de holgura de forma directa, entonces se debe agregar una variable artificial para poder tener una base inicial.

Cuando no se puede encontrar una base inicial factible

El método de la gran M fuerza una solución inicial artificial, penalizándola con un valor muy grande (la constante M) en la función objetivo.

Pasos para realizar el método

1. Escribir el problema en su forma estándar

Convertir todas las restricciones en igualdades:

  • Para restricciones del tipo ≤: Se agrega una variable de holgura.
  • Para restricciones del tipo ≥: Resta una variable y agrega una variable artificial.
  • Para restricciones del tipo =: Agrega una variable artificial.

2. Modificar la función objetivo

Agrega las variables artificiales con un costo muy grande (M):

  • Si es maximización: Resta M ⋅ Ai en la función objetivo, porque queremos que esas variables desaparezcan: penalizando el valor.
  • Si es minimización: Suma M ⋅ Ai.

3. Construir la tabla inicial del método simplex

  • Usa las restricciones en formas de ecuación.
  • Coloca las variables básicas en columnas.
  • Incluye la fila Z (fila de diferencias) con los coeficientes modificados.

4. Aplicar el método simplex

  1. Buscar la variable que entra a la base (La que tiene el valor más negativo en la fila Z)
  2. Buscar la variable que sale de la base (usando la prueba razón mínima).
  3. Pivoteo para obtener la nueva tabla.

5. Repetir iteraciones hasta que no haya más coeficientes negativos en la fila Z

Cuando todos los valores en la Fila Z son mayores a 0, se ha llegado a una solución óptima.

6. Revisar si las variables artificiales están en cero

  • Si todas las variables artificiales son 0 al final, entonces la solución es factible y válida.
  • Si alguna variable artificial queda con valor diferente a 0, el problema no tiene solución factible (es inconsistente con las restricciones).

Ejemplo

Función objetivo:

Minimizar Z = 3x₁ + 2x₂

Sujeto a:

2x₁ - 2x₂ ≥ 3 2x₁ + 2x₂ ≥ 9 x₁, x₂ ≥ 0

Convertir a forma estándar

Z - 3x₁ - 2x₂ - MR₁ - MR₂ = 0 2x₁ - 2x₂ - x₃ + R₁ = 3 2x₁ + 2x₂ - x₄ + R₂ = 9

Tabla inicial

x₁ x₂ x₃ x₄ R₁ R₂ B(j)
Z -3 -2 0 0 -100 -100 0
R₁ 2 -2 -1 0 1 0 3/2
R₂ 2 2 0 -1 0 1 9/2

Nueva fila z = Anterior fila z + (100) * (fila R₁ + fila R₂)

Nueva fila z = (-3, -2, 0, 0, -100, -100, 0) + (100)*(2+2, -2+2, -1+0, 0-1, 1+0, 0+1, 3+9) Nueva fila z = (-3+400, -2+0, 0-100, 0-100, -100+100, -100+100, 0+1200) Nueva fila z = (397, -2, -100, -100, 0, 0, 1200)

Iteración 1

x₁ x₂ x₃ x₄ R₁ R₂ B(j)
Z 397 -2 -100 -100 0 0 1200
R₁ 2 -2 -1 0 1 0 3/2
R₂ 2 2 0 -1 0 1 9/2
x₁ x₂ x₃ x₄ R₁ R₂ B(j)
Z 0 395 197/2 -100 -397/2 0 1209/2
x₁ 1 -1 -1/2 0 1/2 0 3/2
R₂ 0 4 1 -1 -1 1 6

Para fila Z:

x₁ = 397 - (397*1) = 0 x₂ = -2 - (397)(-1) = 395 x₃ = -100 - (397)(-1/2) = 197/2 x₄ = -100 - (397*0) = -100 R₁ = 0 - (397)(1/2) = -397/2 R₂ = 0 - (397*0) = 0 B(j) = 1200-(397)(3/2)=1209/2

Para fila x₁:

x₁ = 2/2 = 1 x₂ = -2/2 = -1 x₃ = -1/2 x₄ = 0/2 = 0 R₁ = 1/2 R₂ = 0/2 = 0 B(j) = 3/2

Para fila R₂:

x₁ = 2 - (2*1) = 0 x₂ = 2 - (2)(-1) = 4 x₃ = 0 - (2)(-1/2) = 1 x₄ = -1 - (2*0) = -1 R₁ = 0 - (2)(1/2) = -1 R₂ = 1 - (2*0) = 1 B(j) = 9-(2)(3/2)=6

Iteración 2

x₁ x₂ x₃ x₄ R₁ R₂ B(j)
Z 0 395 197/2 -100 -397/2 0 1209/2
x₁ 1 -1 -1/2 0 1/2 0 3/2
R₂ 0 4 1 -1 -1 1 6
x₁ x₂ x₃ x₄ R₁ R₂ B(j)
Z 0 0 -1/4 -5/4 -399/4 -395/4 12
x₁ 1 0 -1/4 -1/4 1/4 1/4 3
x₂ 0 1 1/4 -1/4 -1/4 1/4 3/2

Para fila Z:

x₁ = 0 - (395*0) = 0 x₂ = 395 - (395)(1) = 0 x₃ = 197/2 - (395)(1/4) = -1/4 x₄ = -100 - (395)(-1/4) = -5/4 R₁ = -397/2 - (395)(-1/4) = -399/4 R₂ = 0 - (395)(1/4) = -395/4 B(j) = 1209/2-(395)(3/2)=12

Para fila x₁:

x₁ = 1 - (-1*0) = 1 x₂ = -1 - (-1)(1) = 0 x₃ = -1/2 - (-1)(1/4) = -1/4 x₄ = 0 - (-1)(-1/4) = -1/4 R₁ = 1/2 - (-1)(-1/4) = 1/4 R₂ = 0 - (-1)(1/4) = 1/4 B(j) = 3/2-(-1)(3/2)=3

Para fila x₂:

x₁ = 0/4 = 0 x₂ = 4/4 = 1 x₃ = 1/4 x₄ = -1/4 R₁ = -1/4 R₂ = 1/4 B(j) = 6/4 = 3/2

Tabla final

x₁ x₂ x₃ x₄ R₁ R₂ B(j)
Z 0 0 -1/4 -5/4 -399/4 -395/4 12
x₁ 1 0 -1/4 -1/4 1/4 1/4 3
x₂ 0 1 1/4 -1/4 -1/4 1/4 3/2

Solución óptima:

x₁ = 3, x₂ = 3/2, Z = 12
