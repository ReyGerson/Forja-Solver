Documentación del Método Simplex

Inicio

Objetivo del método

Encontrar la solución óptima para un problema de programación lineal, ya sea maximización o minimización a una función de costos, ganancias, producción, estando sujetos a ciertas restricciones. Buscando el mejor punto dentro de un espacio definido con respecto a dichas restricciones que cumplan con el criterio de optimización.

Ventajas

  • Es basado en consideraciones geométricas y no requiere el uso de derivadas de la función objetivo.
  • Es fácil de usar y tiene un alto grado de efectividad.
  • Se puede usar con funciones objetivo ya que las primeras iteraciones se busca el mínimo ampliamente y evitan mínimos fácilmente.
  • Es eficaz y se ajusta a varios parámetros.
  • Garantiza una solución óptima o determina que no hay una solución.

Desventajas

  • Aunque es eficiente para problemas lineales, puede no ser adecuado para problemas con funciones no lineales o restricciones no lineales.
  • La elección inicial de la solución básica puede afectar el rendimiento del método, encontrar una buena solución inicial puede ser difícil en ciertos casos.
  • En problemas muy grandes o complejos el método simplex puede necesitar una cantidad importante de recursos computacionales lo que puede llegar a limitar su aplicación.

Situaciones de uso

Planificación de producción

Determinar la cantidad óptima de diferentes productos a fabricar para maximizar ganancias o minimizar costos.

Gestión de inventario

Decidir los niveles óptimos de inventario para satisfacer la demanda mientras se minimizan los costos de almacenamiento.

Asignación de recursos

Ayuda a asignar presupuesto de manera óptima entre diferentes canales y medios.

Optimización de costos

Encontrar la forma más económica de realizar una tarea o proceso, considerando diferentes variables y restricciones.

Pasos para realizar el algoritmo

1. Definir el problema en la forma estándar y generar matriz

Un programa de programación lineal tiene la siguiente forma:

Función objetivo:

(Maximizar o minimizar)

Z = c₁x₁ + c₂x₂ + ... + cₙxₙ

Sujeto a:

c₁₁x₁ + c₁₂x₂ + ... + c₁ₙxₙ (≥, ≤, =) b₁ c₂₁x₁ + c₂₂x₂ + ... + c₂ₙxₙ (≥, ≤, =) b₂ ... cₘ₁x₁ + cₘ₂x₂ + ... + cₘₙxₙ (≥, ≤, =) bₘ x₁ ≥ 0, x₂ ≥ 0, ..., xₙ ≥ 0

Donde x₁, x₂, ..., xₙ son las variables del problema y luego se verifica que todas las restricciones tienen el lado derecho no negativo, es decir:

b₁ + b₂ + ... + bₘ ≥ 0

Agregar variables de holgura: Convierte las restricciones en ecuaciones.

Ejemplo:

2x₁ + x₂ ≤ 8 → 2x₁ + x₂ + s₁ = 8

2. Construir la tabla inicial del Simplex

Incluye:

  • Variables básicas (VB)
  • Coeficientes de la función objetivo (Z)
  • Coeficientes de las restricciones
  • Resultados del lado derecho (LD)
  • Fila de "Z - C"

3. Identificar la columna pivote (variable entrante)

  • Selecciona el coeficiente más negativo en la fila Z - C. Es la variable que entrará a la base.

4. Identificar la fila pivote (variable saliente)

  • Divide los valores del lado derecho entre los valores positivos de la columna pivote.
  • Elige el menor coeficiente positivo. Esa fila corresponde a la variable que sale de la base.

5. Realizar el pivoteo (actualizar la tabla)

  • Cuando todos los elementos de Z - C sean mayores o iguales a 0 (para maximización), la solución es óptima.
  • El valor óptimo está en la columna LD, correspondiente a Z.
  • Las variables básicas dan su valor en la solución, las no básicas son cero.

Ejemplo

Función objetivo:

Maximizar Z = 3x₁ + 2x₂

Sujeto a:

x₁ + x₂ ≤ 4 x₁ + 3x₂ ≤ 6 x₁, x₂ ≥ 0

Agregar variables de holgura s₁ y s₂

x₁ + x₂ + s₁ = 4 x₁ + 3x₂ + s₂ = 6

Reescribir la función objetivo:

Z = 3x₁ + 2x₂ + s₁ + s₂

Tabla inicial del Simplex

Base x₁ x₂ s₁ s₂ b(j)
s₁ 1 1 1 0 4
s₂ 1 3 0 1 6
z -3 -2 0 0 0

Fila 1: 4/1 = 4
Fila 2: 6/1 = 6

Variable saliente: s₁, Número pivote: 1

Actualizar la tabla (pivoteo)

Base x₁ x₂ s₁ s₂ b(j)
x₁ 1 1 1 0 4
s₂ 0 2 -1 1 2
z 0 1 3 0 12

Nueva fila 1

x₁ = 1/1 = 1 x₂ = 1/1 = 1 s₁ = 1/1 = 1 s₂ = 0/1 = 0 b(j) = 4/1 = 4

Nueva fila 2

x₁ = 1 - (1 × 1) = 0 x₂ = 3 - (1 × 1) = 2 s₁ = 0 - (1 × 1) = -1 s₂ = 1 - (1 × 0) = 1 b(j) = 6 - (1 × 4) = 2

Nueva fila Z

x₁ = -3 + (3 × 1) = 0 x₂ = -2 + (3 × 1) = 1 s₁ = 0 + (3 × 1) = 3 s₂ = 0 + (3 × 0) = 0 b(j) = 0 + (3 × 4) = 12

Resultado final

x₁ = 4, x₂ = 0, Z = 12
